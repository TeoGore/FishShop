Fish Shop - About Us

Shopping Web App

The usage is very simple! The seller needs only o clone the repository of the web app and build his database dependingo on what fish he has. Then it's only a matter to serve the web app through a web server! The web app is written using a modern programming language: Python, with one of its most famous web frameworks: Flask!

Thanks to Flask, the app is built using templating (managed by Jinja engine) so all the redundancy in web pages is removed. This is good for both programmers that need to maintain the web app, and users that receive their content in a fast way

As we speak about the simplicity and minimalism of this web app, it's noticable that in order to run the app it's only needed a python 3 interpreter and some dependecies:

  • Python 3.6 or newer
  • Flask
  • MySQL (client and connector)
  • passlib (optional/still not implemented)

Still we need to improve the application, indeed, we need a good hashing algorithm to hash the passwords of the users (if someone has advices, please contact me!)

Our team is all over the world. Launched in 2019, Fish Shop is a unique web app that enables users to sell their fishes!
